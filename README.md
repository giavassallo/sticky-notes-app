# Sticky Notes App Django Project

# Description:
  Sticky Notes is a web application that allows users to create, update
  view, and delete notes

# Features
  - create notes
  - view all
  - update notes
  - delete note
  - notes in order by creation

# Installation
1. Create virtual environment: python -m venv venv
2. Activate venv: venv\Sripts\activate
3. Install dependencies: pip install -r requirements.txt
4. Apply migrations: python manage.py migrate
5. Run server: python manage.py runserver
